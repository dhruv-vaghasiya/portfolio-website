# 🔐 Dhruv Vaghasiya — Cybersecurity Portfolio Website

A modern, responsive portfolio website built using HTML, CSS, and JavaScript to showcase my cybersecurity skills and projects.

Designed with a futuristic dark theme and smooth animations to reflect my passion for cybersecurity and digital defense.

---

## 🚀 Features

- Modern dark UI with gradient accents  
- Fully responsive design  
- Smooth scroll navigation  
- Animated circular skill progress bars  
- Interactive project cards  
- Glassmorphism-style contact section  
- Direct contact links (Email, LinkedIn, Instagram)  

---

## 🛠️ Built With

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- Vanilla JavaScript  
- Google Fonts (Poppins)  
- DiceBear Avatars API  

---

## 🌍 Deployment (GitHub Pages)

1. Push the project to GitHub  
2. Go to **Settings → Pages**  
3. Select branch: `main`  
4. Save  

Your site will be live at:

```
https://your-username.github.io/repository-name/
```
