# 🔐 Dhruv Vaghasiya — Cybersecurity Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, cybersecurity projects, and contact information.

Designed with a futuristic dark theme and smooth animations to reflect my passion for cybersecurity and digital defense.

---

## 🚀 Live Demo

You can host this project easily using **GitHub Pages**.

---

## 📌 Features

- 🎨 Modern dark UI with gradient accents  
- 📱 Fully responsive design  
- 🔄 Smooth scroll navigation  
- 📊 Animated circular skill progress bars  
- 💡 Interactive project cards with hover effects  
- ✨ Glassmorphism-style contact section  
- 📬 Direct contact links (Email, LinkedIn, Instagram)

---

## 🛠️ Built With

- HTML5  
- CSS3 (Flexbox, Grid, Animations, Custom Properties)  
- Vanilla JavaScript  
- Google Fonts (Poppins)  
- DiceBear Avatars API

3. Click **Pages**
